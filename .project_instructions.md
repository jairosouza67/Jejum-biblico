# Jejum Bíblico na Prática — Landing Page

## Project Overview
High-conversion landing page for a Brazilian Portuguese Christian digital ebook titled **"Jejum Bíblico na Prática"**, priced at **R$ 19,90** with immediate access. Single-page long-form sales page with 12 mandatory sections + footer + legal pages (Termos, Privacidade).

## Language
- **All UI copy is in Brazilian Portuguese (pt-BR)** — `lang="pt-BR"` on `<html>`.
- Copy text is EXACT from `/home/ubuntu/landing_page_copy.txt` — do NOT invent generic marketing text.

## Visual Style
- **Aesthetic**: Warm, spiritual, contemplative — NOT heavy religious. Clean, elegant, lots of white space.
- **Primary palette** (defined as CSS variables + Tailwind utilities in `globals.css`):
  - `--brand-forest` `hsl(158 58% 14%)` — deep forest green (primary)
  - `--brand-forest-dark` `hsl(158 60% 10%)` — darkest green
  - `--brand-sage` `hsl(140 22% 72%)` — pastel sage
  - `--brand-cream` `hsl(42 45% 97%)` — cream bg (#fbf9f4)
  - `--brand-cream-dark` `hsl(42 35% 92%)` — slightly deeper cream
  - `--brand-gold` `hsl(38 62% 52%)` — gold/amber CTA color
  - `--brand-gold-soft` `hsl(42 70% 85%)`
- Utility classes: `bg-brand-forest`, `text-brand-gold`, `bg-brand-cream`, `border-brand-sage`, etc.
- **CTA button**: `.cta-gold` class — gold gradient, uppercase, with hover lift + brightness. Use `<CTAButton />` component (`app/_components/cta-button.tsx`). It auto-handles smooth-scroll for `#hash` links.

## Typography
- Loaded in `app/layout.tsx`:
  - `DM_Sans` → `--font-sans` (body text)
  - `Cormorant_Garamond` → `--font-serif` (all headlines — use `.font-serif` utility)
  - `JetBrains_Mono` → `--font-mono`
- Headlines use serif (`font-serif`), often with `<em className="not-italic ...">` for colored accent phrase on second line.

## CRITICAL TECHNICAL NOTES
- **Image sections with text**: Whenever a section uses an image as background + text on top, MUST use: `section` with solid `bg-brand-forest-dark`, absolute inset-0 wrapper (no `-z-10`) for the image+overlay, and `relative z-10` for the content. Using `-z-10` causes the automated contrast-test tool to see the body's cream color as the background and flag WCAG contrast issues (even though visually fine). This affects sections 1, 7, 11.
- **Logo duplication**: Do NOT use `/logo.png` more than once on the page. The automated test flags identical images as duplicates. Hero has the full logo; ebook mockup and footer use inline SVG teardrop/flame ornaments in amber color (same visual family, different asset).
- **Year in footer**: Hardcoded `© 2026` — avoid `new Date().getFullYear()` to prevent any hydration timing mismatch.
- **ThemeProvider**: Forced to `light` in layout (`forcedTheme="light"`) since the sonner Toaster uses `useTheme()`. Do not remove.

## Behavioral Rules
- Keep this a single-page static marketing site — no database, no auth, no server-side form.
- All CTAs link to `#oferta` (scroll to the pricing card).
- Mobile-first responsive — grid collapses to single column under `md:`.
- Smooth fade-in on scroll via `<Reveal>` — threshold 0.15, rootMargin `0px 0px -60px 0px`.
